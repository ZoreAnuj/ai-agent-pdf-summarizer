# AI-Powered PDF Summarizer

This project is an AI agent that automatically extracts and summarizes key information from PDF documents. It was built to explore intelligent document processing and reduce manual review time for lengthy reports.

## Key Features
- Extracts text from PDFs using OCR and standard parsing
- Generates concise summaries with configurable detail levels
- Preserves critical data points and technical terminology
- Provides structured output in multiple formats

## Tech Stack
- Python
- LangChain
- OpenAI API
- PyPDF2
- FastAPI

## Getting Started
```bash
git clone https://github.com/zoreanuj/ai-agent-pdf-summarizer.git
cd ai-agent-pdf-summarizer
pip install -r requirements.txt
python app.py
```
